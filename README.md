# YouTube Embed plugin for Typesetter CMS #

## About
* This plugins provides a new section type to embed YouTube videos
* It supports responsive scaling for 16:9 (default) and 4:3 aspect ratios and fixed video size

Basic usage: Just paste your YouTube embed code into the editor, click [Apply] and you’re all set.

See also [WIP plugin page](https://typesetter-addons.grafikrausz.at/YouTube-Embed), [Typesetter Home](https://www.typesettercms.com), [Typesetter on GitHub](https://github.com/Typesetter/Typesetter)

## Requirements ##
* Typesetter CMS 5.0+


## Manual Installation ##
1. Download the [master ZIP archive](https://github.com/juek/YouTubeEmbed/archive/master.zip)
2. Upload the extracted folder 'YouTubeEmbed-master' to your server into the /addons directory
3. Install using Typesetter's Admin Toolbox &rarr; Plugins &rarr; Manage &rarr; Available &rarr; YouTubeEmbed
